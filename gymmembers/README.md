# About Dataset

Este conjunto de datos proporciona una visión detallada de las rutinas de ejercicio de los miembros de un gimnasio, sus atributos físicos y sus métricas de condición física. Contiene **973 registros** de datos de gimnasio, incluyendo indicadores clave de rendimiento como la frecuencia cardíaca, las calorías quemadas y la duración de las sesiones de entrenamiento.

Cada registro también incluye información demográfica y el nivel de experiencia del usuario, lo que permite realizar análisis completos sobre patrones de entrenamiento, progresión física y tendencias relacionadas con la salud.

## Características principales

| Variable | Descripción |
|----------|-------------|
| **Age** | Edad del miembro del gimnasio. |
| **Gender** | Género del miembro (Male o Female). |
| **Weight (kg)** | Peso del miembro en kilogramos. |
| **Height (m)** | Estatura del miembro en metros. |
| **Max_BPM** | Frecuencia cardíaca máxima (latidos por minuto) durante el entrenamiento. |
| **Avg_BPM** | Frecuencia cardíaca promedio durante la sesión de entrenamiento. |
| **Resting_BPM** | Frecuencia cardíaca en reposo antes del entrenamiento. |
| **Session_Duration (hours)** | Duración de cada sesión de entrenamiento en horas. |
| **Calories_Burned** | Total de calorías quemadas durante la sesión. |
| **Workout_Type** | Tipo de entrenamiento realizado (por ejemplo: Cardio, Strength, Yoga, HIIT). |
| **Fat_Percentage** | Porcentaje de grasa corporal del miembro. |
| **Water_Intake (liters)** | Cantidad de agua consumida durante el entrenamiento (litros). |
| **Workout_Frequency (days/week)** | Número de sesiones de entrenamiento por semana. |
| **Experience_Level** | Nivel de experiencia, desde principiante (1) hasta experto (3). |
| **BMI** | Índice de Masa Corporal (IMC), calculado a partir del peso y la estatura. |

## Aplicaciones del dataset

Este conjunto de datos es ideal para:

- Ciencia de datos y análisis predictivo.
- Investigación en salud y actividad física.
- Estudios sobre hábitos de entrenamiento.
- Modelado de la progresión física de los atletas.
- Análisis de la relación entre variables demográficas, fisiológicas y de rendimiento.
- Desarrollo de modelos de machine learning relacionados con fitness y bienestar.

Gracias a la diversidad de sus variables, este dataset permite obtener información valiosa sobre cómo distintos factores influyen en la intensidad del entrenamiento, la resistencia física y la salud general.
